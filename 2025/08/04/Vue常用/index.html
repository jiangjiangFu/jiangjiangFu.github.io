Vue 3 工作常用笔记

✅ 推荐使用 Composition API，搭配 <script setup> 书写更简洁。

🌀 生命周期函数（含解释 + 示例）

Vue 的生命周期函数允许你在组件的不同阶段执行特定操作，比如请求数据、销毁前清理资源等。

生命周期Composition API 示例说明
挂载后 ✅onMounted(() => { console.log('mounted') })DOM 已挂载，常用于请求数据、初始化操作
卸载后 🧹onUnmounted(() => { cleanup() })组件卸载时执行，如移除监听器、清理定时器等
更新后onUpdated(() => { console.log('DOM更新') })数据响应更新后，DOM 变更完毕后调用

🔗 组件通信常用方式

1. 父传子（props）

父组件通过 props 传递数据到子组件。

2. 子传父（emit）

子组件通过事件将数据或行为通知父组件。

3. 全局状态（Pinia）

用于跨组件共享状态。

📌 常用指令示例

Vue 指令是模板语法的核心，用于控制 DOM 行为。

⚙️ Composition API 常用函数

Composition API 提供灵活的响应式和逻辑组合方式。

🛣️ Vue Router 4 示例

Vue Router 负责页面路由跳转。

🧩 Pinia 简单示例（状态管理）

Pinia 是 Vue 官方推荐的状态管理工具，代替 Vuex。

🧰 常用工程化建议

  • ✅ 使用 defineAsyncComponent 实现组件懒加载，提升首屏速度。
  • ✅ 避免深层响应式数据导致性能问题，使用 shallowReactive。
  • ✅ 封装 axios 请求，统一错误处理、Token 注入。
  • ✅ 模块化目录结构，提升维护性和可读性。

✅ 表单验证（VeeValidate 示例）

✅ API 封装（axios 示例）

✅ 性能优化建议

  • 懒加载路由与组件：defineAsyncComponent, 路由 component: () => import(...)
  • 虚拟滚动列表：vue-virtual-scroller 等插件适用于大量列表渲染
  • 防抖/节流：避免频繁事件触发（如 input、scroll）
  • 图片优化：使用 lazy 属性，压缩资源大小
  • 缓存组件：使用 <KeepAlive> 包裹需要缓存的组件
