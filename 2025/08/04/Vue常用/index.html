🌀 生命周期函数

Vue 的生命周期函数允许你在组件生命周期的特定节点执行代码，通常用于初始化、请求数据、清理资源等操作。

生命周期Composition API 示例说明
挂载后 ✅onMounted(() => { ... })DOM 已挂载，请求数据/初始化
卸载后 🪃onUnmounted(() => { ... })组件卸载时，删除监听器/清理
更新后onUpdated(() => { ... })DOM 更新后调用
